# YouTube Video Downloader

A simple Python project to download YouTube videos using yt-dlp.

## Features

- Download YouTube videos
- Download highest resolution automatically
- Beginner-friendly Python code
- Simple command-line interface

## Technologies Used

- Python
- yt-dlp

## Installation

Install required library:
pip install yt-dlp

## Run the Project

python code.py

## Example

Enter YouTube URL:
https://youtu.be/example
